\begin{DoxyItemize}
\item Click the {\bfseries{Fork}} button on the top right of the \href{https://github.com/liltymoor/stash-vcs}{\texttt{ repository page}}.
\item Clone your forked repository to your local machine\+: {\ttfamily bash git clone \href{https://github.com/YOUR-USERNAME/stash-vcs.git}{\texttt{ https\+://github.\+com/\+YOUR-\/\+USERNAME/stash-\/vcs.\+git}} }

\item Follow the project\textquotesingle{}s coding style and conventions.

\begin{DoxyItemize}
\item Ensure your changes work as expected.
\item If applicable, add or update tests to cover your changes.
